# Code of Conduct

TON VS Code Extension is an open source project. We want the repository to stay
useful, technical, and respectful for contributors, users, and maintainers.

## Expected Behavior

- Be respectful and professional in issues, pull requests, discussions, and
  other project spaces.
- Focus criticism on ideas, code, and behavior, not on people.
- Assume good faith, ask for clarification when needed, and prefer precise,
  actionable feedback.
- Keep discussion relevant to the project and avoid unnecessary hostility or
  escalation.

## Unacceptable Behavior

- Harassment, intimidation, threats, or personal attacks.
- Discriminatory, abusive, or deliberately inflammatory language.
- Repeated bad-faith arguing, trolling, or derailing technical discussions.
- Publishing someone else's private or sensitive information without consent.
- Spam, impersonation, or other disruptive behavior that makes collaboration
  harder.

## Scope

This policy applies to project spaces such as GitHub issues, pull requests,
code review, discussions, and other official project communication channels.

## Enforcement

Project maintainers may edit, hide, lock, or remove content that violates this
policy. They may also restrict participation when necessary to keep the project
safe and productive.

If you see behavior that violates this Code of Conduct, contact the maintainers
through the repository's official GitHub channels.
